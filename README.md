# STOCK_TICKER
# NAME :- TEJASHREE H
## DESCRIPTION :- This Python program lets you check real-time stock prices using easy commands.You can enter a company’s stock symbol, and the script will show the latest price, opening price, day high, day low, and more.It’s simple, lightweight, and perfect for beginners who want to learn how APIs work in Python.

# Features

* Fetches live stock prices

* Shows basic stock details

* Clean and beginner-friendly code

* Works with any stock symbol

* Can be extended for charts or alerts
